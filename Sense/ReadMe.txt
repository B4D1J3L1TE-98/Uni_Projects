Run the executable file called 'Sense_Build.exe' and press Play. 

The screen resolution shouldn't affect the game, nor should the graphics quality.
Select Monitor should be set to Display 1, which is the default and the only display 
used in the game.

Controls (explained in the in-game tutorial also)
Movement ----------------------------- WASD or arrow keys
Look --------------------------------- Mouse
Sprint ------------------------------- Left Shift
Jump --------------------------------- Space Bar
Open/Close eyes ---------------------- E

reset level -------------------------- R   (Handy if an unexpected bug should happen to occur)
exit tutorial ------------------------ Q
